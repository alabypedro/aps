Nós somos um grupo formado por 4 estudantes, pertencentes à UNIP (Universidade Paulista), que tínhamos como objetivo criar um site utilizando a linguagem de marcação HTML, a linguagem de estilo CSS e a linguagem de programação Java Script. Realizamos algumas reuniões via Discord (discord.com) e a partir dessas reuniões decidimos criar um site que calcula a taxa de metabolismo basal, com o objetivo de saber quantas calorias precisávamos ingerir por dia e uma calculadora IMC (Índice de Massa Corpórea) com o objetivo de descobrir o valor do IMC de um determinado indivíduo, e assim, classifica-lo de acordo com o grau de obesidade, variando de 0 a III. Dentro do nosso site, precisávamos inserir 5 imagens, 5 links e 5 páginas para estar de acordo com o propósito de criação do nosso site.
